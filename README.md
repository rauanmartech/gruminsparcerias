## Projeto GRUMINS – Landing Page

Este repositório contém a landing page da **GRUMINS | Grupo Mineiro de Negócios**, focada em parcerias estratégicas no mercado imobiliário de alto padrão.

## Tecnologias utilizadas

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Requisitos

- Node.js e npm instalados

## Como rodar o projeto localmente

```sh
# Instalar dependências
npm install

# Rodar em modo desenvolvimento
npm run dev

# Build para produção
npm run build

# Pré-visualizar o build
npm run preview
```

## Estrutura básica

- `src/` – código-fonte da aplicação (componentes, estilos, lógica)
- `public/` – arquivos estáticos públicos
